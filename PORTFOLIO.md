# コンビニAR - ポートフォリオ

---

## どんなサービスか

**「カメラをかざすだけで、目の前がコンビニになる」モバイルWeb ARアプリ。**

スマホのカメラで身の回りのものを映すと、AIがペットボトルやカップ麺、本などの「コンビニにありそうな商品」をリアルタイムで検出。検出された物体にはコンビニ風の値札が出現し、検出数が増えるにつれて画面全体が蛍光灯・商品棚・レジ・おでんケースなどで彩られ、本物のコンビニ空間に変身していきます。

さらに、人物を検出すると服の色を解析してセブン-イレブン・ローソン・ファミリーマートのいずれかを判定し、その場でコンビニ店員に変身。値札を店員にドラッグ＆ドロップすれば、PayPayモック決済で「購入体験」まで完結します。

---

## 推しアイデア

**「どこでもコンビニ化」という一発で伝わるコンセプト。**

日本に55,000店以上あるコンビニは、誰もが知っている日常的な空間。その「コンビニらしさ」をAIで検出して、リアルな空間をコンビニに変えるという体験は、デモした瞬間に「面白い！」と伝わります。

特に面白いのは：
- **服の色でブランド判定** — 赤い服を着ていればセブン、青ならローソン、緑ならファミマの店員に変身
- **段階的変身** — 1つ検出するごとに少しずつコンビニ化していく「じわじわ感」
- **AR上で買い物** — 値札を店員にドラッグ＆ドロップ → PayPay決済 → レシートという、物理と仮想が融合した購入体験
- **ブランド別入店音** — セブン、ファミマの入店音がブランド判定に連動して鳴る

---

## 作った背景

ハッカソンで「ARで何か面白いことをやりたい」というお題からスタート。

「カメラ越しに何かを認識してリアクションする」というARの基本パターンに対して、「認識した結果に応じて世界そのものを変える」というアプローチを考えました。

コンビニを選んだ理由は：
1. **認識対象が身の回りに溢れている** — ペットボトル、本、スマホ、傘など、COCO-SSDが検出できるカテゴリとコンビニ商品の重なりが多い
2. **空間演出の要素が豊富** — 蛍光灯、棚、レジ、おでんなど、「コンビニっぽさ」を構成するパーツが明確で、段階的に出せる
3. **誰でも体験価値がわかる** — コンビニは全員が知っている空間なので、「確かにコンビニっぽい！」という共感が得やすい

## 推し技術

### 1. ブラウザ完結のリアルタイム物体認識
TensorFlow.js + COCO-SSD をブラウザ上で直接実行。ネイティブアプリ不要、インストール不要。URLを開くだけでAR体験ができます。`lite_mobilenet_v2` モデルで軽量化し、400ms間隔の検出でモバイルでもスムーズに動作。

### 2. Canvas APIによる服の色解析 → ブランド判定
人物検出時に、bboxの胴体部分をCanvasに描画してピクセルサンプリング。RGB→HSL変換し、色相で3ブランドに分類する独自アルゴリズム。カメラ映像からリアルタイムで色を取得するため、着替えれば別ブランドの店員に変わります。

### 3. 全AR演出をSVG + CSSで描画（画像アセットゼロ）
エプロン（肩紐・腰紐・ポケット・ロゴ）、商品棚（6段×5商品のシルエット）、レジ（LCD画面・NFCリーダー）、おでん鍋（SVGアニメ湯気）など、全ての視覚要素をインラインSVGとCSSで構築。外部画像への依存がなく、高解像度でもシャープに表示されます。

### 4. Touch EventsによるAR上ドラッグ＆ドロップ購入
AR空間上の値札をタッチドラッグし、物体認識で検出された人物（店員）のbboxにドロップすることでカートに追加。検出座標とタッチ座標のヒットテストでドロップ判定を行い、物理世界のオブジェクトとARインタラクションを融合させています。

## プロジェクト概要

| 項目 | 内容 |
|------|------|
| **プロジェクト名** | コンビニAR |
| **URL** | https://hackathonar.vercel.app |
| **GitHub** | https://github.com/nakedtatsuya/konbini_ar |
| **開発期間** | 1日（ハッカソン） |
| **チーム** | 個人開発 |
| **カテゴリ** | AR / AI / モバイルWeb |

### 技術スタック

| レイヤー | 技術 |
|----------|------|
| フレームワーク | Next.js 16 (App Router) + TypeScript |
| スタイリング | Tailwind CSS |
| AI/物体認識 | TensorFlow.js + COCO-SSD (lite_mobilenet_v2) |
| 色解析 | Canvas API + HSL色空間分類 |
| AR描画 | インラインSVG + CSS (画像アセットゼロ) |
| サウンド | HTMLAudioElement (ブランド別3種) |
| インタラクション | Touch Events API (ドラッグ＆ドロップ) |
| デプロイ | Vercel |

### 主要機能

- リアルタイム物体認識（30種のコンビニ商品マッピング）
- コンビニ度スコア（0-100%）に応じた12段階のAR空間演出
- 服の色解析によるセブン/ローソン/ファミマのブランド判定
- SVGベースのコンビニ店員ユニフォーム（エプロン・名札・吹き出し）
- ブランド別入店サウンドエフェクト
- AR上のドラッグ＆ドロップ購入フロー
- PayPayモック決済（QRスキャン演出 + レシート発行）

## サムネイル画像について

サムネイル用のOGP画像は、以下の方法で作成できます：

1. **スクリーンショット**: 実機でアプリを起動し、コンビニ度50%以上の状態をスクショ
2. **推奨構図**: 画面に商品棚・値札・店員ユニフォーム・レジが写っている状態
3. **OGPページ**: `/og` ルートを作成してVercel OG Image Generationで自動生成も可能
